      ประวัติคำถามและคำตอบทั้งหมด

      ยังไม่ได้ตั้งค่า API Key

      กรุณาใส่ Gemini API Key ในไฟล์ .env ก่อนเริ่มใช้งาน

      1. ไปที่ Google AI Studio → สร้าง API Key

      2. Copy key ที่ได้ (ขึ้นต้นด้วย AIza...)

      3. เปิดไฟล์ .env แล้วใส่: GEMINI_API_KEY=AIza...

      4. Refresh หน้าเว็บ (ไม่ต้อง restart server)

      ⚠️ ต้องเปิดเว็บผ่าน node server.js (port 3000) ไม่ใช่ live-server

      วิธีใช้งาน

      1

      ตั้งค่า API Key

      เปิดไฟล์ .env แล้ววาง Gemini API Key ที่ได้จาก Google AI Studio ลงในรูปแบบ GEMINI_API_KEY=your_key

      2

      กรอกข้อมูลและเลือกตัวเลือก

      กรอกชื่อ ระดับการศึกษา สาขาวิชา ประสบการณ์ และเลือกตำแหน่งงาน ระดับความยาก รูปแบบ และจำนวนคำถาม

      3

      เข้าสู่ห้องสัมภาษณ์

      AI จะพูดคำถามออกมาเป็นเสียง เมื่อ AI พูดเสร็จ กดปุ่มไมโครโฟนแล้วพูดตอบ หรือพิมพ์คำตอบแทนได้

      4

      ตรวจสอบและส่งคำตอบ

      ระบบจะแปลงเสียงเป็นข้อความ ตรวจสอบให้แน่ใจว่าถูกต้องแล้วกด "ส่งคำตอบ" หากไม่ถูกต้อง กด "ลองพูดใหม่" ได้

      5

      ดูผลการประเมิน

      เมื่อสัมภาษณ์ครบ ระบบจะแสดงคะแนน จุดแข็ง จุดที่ควรปรับปรุง คำแนะนำ และสามารถดาวน์โหลดรายงานได้

      แนะนำให้ใช้ Google Chrome เพื่อการรับเสียงที่ดีที่สุด
